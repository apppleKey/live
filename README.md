# live
`一个关于视屏直播的项目`
# 开始
1. 克隆项目
`$ git clone https://github.com/apppleKey/live.git`
2. 切换目录
`$ cd live`
3. 安装依赖
`$ npm i`
4. 开启服务
`$ npm run dev`
5. 打包生产
`$ npm run build`

# 项目说明
 `1.视屏流

 rtmp:清晰度高 只有flash兼容 苹果不能播 flv格式 

hls:延迟10-30秒 苹果浏览器可以播 主要使用HTML5直播技术 

 `
 `2.技术栈
 vue+vueRouter
 主要使用videojs插件进行播放做到各端浏览器兼容的目前在研究阶段
 `
